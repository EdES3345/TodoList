# TodoList

Une simple application de listes de tâches avec un frontend en React avec TypeScript et un backend avec express et une base de données avec SQLite.

## Lancer le projet en mode stable (production)

1. Installez [NodeJS](https://nodejs.org/fr/download)

2. Téléchargez le dossier du projet et ouvrez-le dans un terminal

3. Lancez l'app React : 
    - `npm i` 
    - `npm run build`
    - `npm run preview`

4. Cliquez sur le lien local pour accéder à l'application

5. Ouvrez un nouveau terminal de nouveau dans le dossier du projet

6. Lancez le serveur:
    - `cd server`
    - `npm  i`
    - `npm start`

## Lancer le projet en mode développement

1. Installez [NodeJS](https://nodejs.org/fr/download)

2. Téléchargez le dossier du projet et ouvrez-le dans un terminal

3. Lancez l'app React : 
    - `npm i` 
    - `npm run dev`

4. Cliquez sur le lien local pour accéder à l'application

5. Ouvrez un nouveau terminal de nouveau dans le dossier du projet

6. Lancez le serveur:
    - `cd server`
    - `npm  i`
    - `npm run dev` (Nodemon est lancé pour redémarrer le serveur à chaque modification)
