# TodoList

Une simple application de listes de tâches avec une gestion de comptes utilisateurs via un serveur.
Ce logiciel est composé d'un frontend en React avec TypeScript et un backend avec express et une base de données avec SQLite.

## Lancer le projet en mode stable (production)

1. Installez [NodeJS](https://nodejs.org/fr/download)

2. Téléchargez le dossier du projet et ouvrez-le dans un terminal

3. Lancez l'app React :
    ```bash
    npm i
    npm run build
    npm run preview
    ```

5. Cliquez sur le lien local pour accéder à l'application

6. Ouvrez un nouveau terminal de nouveau dans le dossier du projet

7. Lancez le serveur:
    ```bash
    cd server
    npm  i
    npm start
    ```

## Lancer le projet en mode développement

1. Installez [NodeJS](https://nodejs.org/fr/download)

2. Téléchargez le dossier du projet et ouvrez-le dans un terminal

3. Lancez l'app React :
    ```bash 
    npm i 
    npm run dev
    ```
    
5. Cliquez sur le lien local pour accéder à l'application

6. Ouvrez un nouveau terminal de nouveau dans le dossier du projet

7. Lancez le serveur:
   ```bash
   cd server
   npm  i
   npm run dev # Nodemon sera lancé pour redémarrer le serveur à chaque modification
   ```
